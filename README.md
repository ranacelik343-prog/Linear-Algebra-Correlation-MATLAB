# Linear Algebra Correlation Analysis (MATLAB)

This project demonstrates the calculation of the Pearson Correlation Coefficient using fundamental Linear Algebra concepts in MATLAB, without relying on built-in statistical functions. 

## 🚀 Features & Technical Details
* **Data Centering:** Translating vectors to the origin by subtracting the mean.
* **Matrix Operations:** Calculation of the dot product matrix (Z'Z) and vector norms.
* **Mathematical Modeling:** Generation of a Pearson Correlation Matrix entirely from scratch.
* **Data Visualization:** Displaying the correlations using a custom, high-contrast Heatmap in MATLAB.

## 🛠️ Technologies Used
* MATLAB
* Linear Algebra (Vectors, Norms, Inner Products)
* Data Analysis
